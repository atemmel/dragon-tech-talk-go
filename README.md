# dragon-tech-talk-go

Slides i markdownformat om programmeringsspråket Go

## Innehåll

Slides går enklast att se i Github, se mappen namngiven `slides`

## Presentationsvisning

Kräver [kraftpek](https://github.com/atemmel/dtt-kraftpek) för att visas som en presentation

Efter att kraftpek är installerat kan du visa slides genom
```sh
git clone https://github.com/atemmel/dragon-tech-talk-go

cd dragon-tech-talk-go

kraftpek ./slides
```
